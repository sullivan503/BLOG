# 🚀 疯文斋：从代码到上线的最后一步

你好！这是你的网站项目。因为浏览器无法直接读取 `.tsx` 源代码，我们需要通过一个“打包”过程将它变成浏览器能理解的网页。

## 📥 第一步：下载代码
点击 Google AI Studio 界面上的下载按钮，将整个项目下载到你的电脑并解压。

## 💻 第二步：准备本地环境 (仅需执行一次)
1. **安装 VS Code**: [前往官网下载](https://code.visualstudio.com/)
2. **安装 Node.js**: [前往官网下载](https://nodejs.org/) (选择 LTS 版本)

## 🏗️ 第三步：打包网站
1. 用 VS Code 打开你解压后的文件夹。
2. 按快捷键 `Ctrl + ~` (Windows) 或 `Cmd + ~` (Mac) 打开底部的**终端 (Terminal)**。
3. 在那个黑窗里依次输入以下两条命令（每行输完按回车）：
   ```bash
   npm install
   npm run build
   ```
4. 等待完成后，你会发现文件夹里多了一个 **`dist`** 文件夹。这就是我们要的“成品”。

## ☁️ 第四步：上传到宝塔
1. 登录你的宝塔面板，进入文件管理器。
2. 进入目录 `/www/wwwroot/74.48.186.237`。
3. **重要**：将原有的 `index.php` 改名为 `index_backup.php`。
4. 点击“上传”，将你本地 **`dist` 文件夹里面**的所有内容（index.html 和 assets 文件夹）拖进去。

## ⚙️ 第五步：设置伪静态 (防止刷新 404)
在宝塔面板的网站设置里，找到“伪静态”，粘贴以下代码：
```nginx
location / {
    try_files $uri $uri/ /index.html;
}
```
